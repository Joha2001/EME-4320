Hello! My name is John Hoang. This website is being developed for the class, EME 4320: Instructional Development for Teaching and Learning. This page is for the first project.
The goal of the project is to display a basic understanding of HTML AND CSS.

Using Paragraphs (<p> </p>)

This project requires an individual to use multiple different tags and styles.
The different possible tags and styles will be shown using lists below.

Using Lists (<ul> </ul>) and (<ol> </ol>)

Unordered List

  • One can bold (<b> </b>) text.
  • One can italicize (<i> </i>) text
  • One can subscript (<sub> </sub>) text
  • One can superscript (<sup> </sup>) text
  • One can delete (<del> </del>) text
  • One can mark (<mark> </mark>) text

Ordered List

  1. One can change the font. (Using style="font-family:'Courier New';)
  2. One can change the font color. (<font color="blue"> </font>)
  3. One can change the font size.
